# Исследование графических интерфейсов и событий в Python с использованием Tkinter

## Описание работы

Данная работа посвящена изучению основ создания графических интерфейсов на языке Python с использованием библиотеки Tkinter. Tkinter позволяет создавать окна, рисовать графические объекты и обрабатывать события пользователя, такие как нажатия мыши и клавиатуры.

В ходе выполнения работы были рассмотрены несколько примеров применения Tkinter: создание графического окна, рисование простых фигур, обработка событий мыши, движение объектов и управление ими с клавиатуры.

---

# Цель работы

Изучить принципы создания графических приложений на Python и реализовать несколько практических примеров с использованием библиотеки Tkinter.

---

# Задачи работы

Для достижения поставленной цели были выполнены следующие задачи:

- изучить создание графических окон с использованием Tkinter;
- реализовать рисование простых графических фигур, таких как прямоугольники и круги;
- изучить обработку событий мыши для интерактивного взаимодействия;
- реализовать движение графического объекта по экрану с использованием таймера;
- реализовать управление объектом с клавиатуры, обеспечивая перемещение по экрану;
- закрепить навыки работы с координатами, цветами и размерами объектов в графическом окне.

---

# Используемые технологии

В процессе выполнения работы использовались следующие технологии:

- язык программирования **Python**
- библиотека **Tkinter** для создания графического интерфейса

---

# Создание графического окна

Первое задание демонстрирует создание графического окна с определёнными размерами и заголовком. Окно создаётся с фиксированными размерами и является базой для дальнейшего отображения графических объектов и обработки событий.

---

# Рисование простых фигур

Второе задание позволяет рисовать простые графические фигуры, такие как прямоугольники и круги, с заданными цветами и рамками. Это позволяет изучить работу с координатами, размерами и цветами объектов на графическом холсте.

---

# Обработка событий мыши

Третье задание показывает, как обрабатывать события мыши. При клике пользователем на холст программа определяет координаты точки клика и рисует в этой точке круг случайного цвета. Данный пример демонстрирует интерактивность и реакцию программы на действия пользователя.

---

# Движение объекта

Четвёртое задание демонстрирует движение объекта по экрану с помощью таймера. Прямоугольник постепенно перемещается по горизонтали, а программа проверяет границы окна, чтобы объект не выходил за пределы холста. Это позволяет изучить основы анимации в Tkinter.

---

# Управление объектом с клавиатуры

Пятое задание реализует управление объектом с помощью клавиш стрелок. Пользователь может перемещать прямоугольник по экрану в разные направления, при этом программа проверяет границы окна. Данный пример демонстрирует обработку событий клавиатуры и управление положением объектов в графическом интерфейсе.

---

# Вывод

В ходе выполнения данной работы были изучены основные принципы работы с графическим интерфейсом в Python и библиотекой Tkinter:

- создание графических окон и холстов для отображения объектов;
- рисование фигур с заданными координатами, размерами и цветами;
- обработка событий мыши и клавиатуры для интерактивного взаимодействия;
- перемещение и анимация объектов на экране.

Реализованные задания позволяют понять основы создания графических приложений и взаимодействия пользователя с программой, а также закладывают базу для более сложных графических проектов.
